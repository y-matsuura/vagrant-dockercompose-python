# PyCharm で Vagrant 上の python のインタープリタを指定する
・File > Settings を選択
・Project Interpreter を選択
・設定アイコン > Add を選択
・Vagrant を選択
・Python interpreter path を /usr/bin/python3 とする
[参考]
https://pleiades.io/help/pycharm/configuring-remote-interpreters-via-virtual-boxes.html
